# Parrot Stand Up

Asynchronous Stand Up Slack Bot

## Setup

### Environment variables

Set up environment variables on `.env` following `.env.example`

### Install & Run

```shell
pipenv install
```

```.shell
pipenv shell
```

```.shell
pipenv run python app.py
```
